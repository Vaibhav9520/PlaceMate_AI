3. **Create account** with email/password
4. **Verify email** if required
5. **Go to**: https://huggingface.co/settings/tokens
6. **Click "New token"**
7. **Name**: "PlaceMate AI Interview"
8. **Role**: Select "Read" (default is fine)
9. **Click "Generate a token"**
10. **Copy the token** (starts with `hf_...`)
