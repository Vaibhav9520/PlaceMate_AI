🚀 PlaceMate AI - Start Guide

Follow these simple steps to start your application

📌 Quick Info: You need to start TWO servers - Backend (API) and Frontend (React App)

1Easiest Way - Automatic Start

Windows Users:

  1. Find start.bat file in your project folder
  2. Double-click it
  3. Two terminal windows will open automatically
  4. Wait 10-15 seconds

Mac/Linux Users:

chmod +x start.sh
./start.sh

2Manual Way - Step by Step

Terminal 1 - Start Backend:

cd server
npm run dev

You should see:

Simple File Database Connected
Data stored in: C:\...\server\data
Server running in development mode on port 5000

Terminal 2 - Start Frontend:

cd client
npm run dev

You should see:

VITE v6.4.1 ready in 290 ms
➜ Local: http://localhost:5173/

3Open in Browser

Once both servers are running, open your browser and go to:

🌐 Open PlaceMate AI ✓ Check Backend Status

✅ Success Checklist

  • Backend server started (Terminal 1 shows "Server running on port 5000")
  • Frontend client started (Terminal 2 shows "Local: http://localhost:5173/")
  • No error messages in either terminal
  • Browser opens http://localhost:5173
  • Landing page loads with "PlaceMate AI" title
  • Can see "Sign Up" and "Sign In" buttons
⚠️ Common Issues:
  • Port already in use: Close other applications using ports 5000 or 5173
  • Module not found: Run npm install in both server and client folders
  • Blank page: Make sure backend is running first, then start frontend

🛑 How to Stop Servers

When you're done:

  1. Go to each terminal window
  2. Press Ctrl+C
  3. Or simply close the terminal windows
