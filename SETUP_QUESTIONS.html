🚀 Setup Steps

1 Generate Questions (One-Time)
generate-questions.bat

This will use Gemini AI to generate 10 questions for each of 15 topics:
• Arrays, Strings, Linked Lists
• Stacks, Queues, Trees, Graphs
• Sorting, Searching, Dynamic Programming
• Recursion, Backtracking, Greedy, Hash Tables, Heaps

⏱️ Time: 5-10 minutes (150 questions total)

2 Seed MongoDB (Optional)
seed-questions.bat

Upload questions to MongoDB Atlas for production use.
Skip this if: You're using file-based storage (USE_MONGODB=false)

3 Restart Server
restart-server-now.bat

Server will now use pre-generated questions!

💾 Storage Options

Option 1: MongoDB Atlas (Production)

• Fast database queries
• Scalable for many users
• Requires MongoDB setup
• Set USE_MONGODB=true

Option 2: File-Based (Development)

• Loads from JSON file
• No database needed
• Perfect for testing
• Set USE_MONGODB=false
