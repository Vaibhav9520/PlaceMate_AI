🔧 Clear Browser Data

Why do I need to do this?

Your browser has stored an old authentication token that's incompatible with the current database. Clearing it will allow you to log in fresh.

Step 1: Click the button below to clear stored data
✅ Browser data cleared successfully!
Step 2: After clearing, go to the application
Manual Method (if button doesn't work):
  1. Press F12 to open Developer Tools
  2. Go to the Application tab
  3. Click Local Storage in the left sidebar
  4. Click on http://localhost:5174
  5. Right-click and select Clear
  6. Refresh the page
