🚀 Quick Start

1 Start the Server
restart-server-now.bat

Wait for: Server running in development mode on port 5000

2 Access the Feature

1. Open http://localhost:5173
2. Login to your account
3. Go to Dashboard
4. Click "DSA Coding Practice" (purple card)

3 Start Coding!

1. Select a DSA topic (Arrays, Strings, Trees, etc.)
2. Choose number of questions
3. Click "Generate Questions"
4. Solve problems and run your code!

📖 How It Works

Workflow

1. Topic Selection → Choose DSA topic and question count
2. AI Generation → Gemini creates unique problems
3. Problem Display → Read description, examples, constraints
4. Code Writing → Write solution in preferred language
5. Code Execution → Run against test cases
6. Results → See which tests passed/failed
7. Navigation → Move to next question or retry
