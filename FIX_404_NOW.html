🔧 Fix 404 Error

Coding Practice Not Working

❌ The Error

POST http://localhost:5000/api/coding/generate-questions
404 (Not Found)

✅ The Problem

1. Missing axios dependency - Server needs axios to work

2. Server not restarted - New routes need restart to load

🚀 Quick Fix

Run this command in your terminal:

setup-coding-practice.bat

This will automatically:

1 Stop the server
2 Install axios dependency
3 Restart the server
4 Test the endpoints
✓ Show you it's working!

📋 Manual Fix

Step 1: Install Axios

cd server
npm install axios

Step 2: Restart Server

taskkill /F /IM node.exe
(wait 5 seconds)
cd server
npm run dev

Step 3: Test

node test-coding-endpoint.js

Expected Output:

✅ Server is running
✅ Coding endpoint exists (returns 401 Unauthorized)

🎉 After Fix

1 Open http://localhost:5173
2 Login to your account
3 Click "DSA Coding Practice" on Dashboard
4 Select a topic (e.g., Arrays)
5 Click "Generate Questions"
✓ Questions generated! Start coding! 🎉

❓ Why This Happened

The codingService.js file uses axios to call external APIs:

import axios from 'axios';

But axios wasn't installed in server/package.json, so the server couldn't load the routes.

Ready to Fix!

Run setup-coding-practice.bat in your terminal
